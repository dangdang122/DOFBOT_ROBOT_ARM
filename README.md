# 🤖 DOFBOT-IK-Workspace
> **6축 로봇 팔(DOFBOT)의 역운동학 제어 및 작업 공간 분석 프로젝트입니다.**

---

## 📌 1. 프로젝트 개요
이 프로젝트는 **Yahboom DOFBOT**을 정밀하게 제어하기 위한 알고리즘 모음입니다. 
단순한 각도 제어를 넘어, 수학적 계산을 통해 공간 좌표($x, y, z$)로 로봇을 움직이고 안전하게 운용할 수 있도록 돕습니다.



---

## 🛠 2. 하드웨어 제원 (Hardware Spec)
로봇의 물리적 수치는 계산의 정확도를 위해 아래와 같이 설정되었습니다.

## 📂 3. 주요 파일 및 기능 설명

### 🟢 `IK.py` (Inverse Kinematics)
로봇의 **실질적인 움직임**을 담당합니다.
* **삼각함수 기반 IK**: 코사인 법칙을 이용하여 $L2$, $L3$의 관절 각도를 계산합니다.
* **최적 피치 탐색**: 서보 모터 가동 범위(0~180도) 내에서 최적의 해를 찾습니다.
* **선형 보간**: 지점과 지점 사이를 부드러운 직선으로 이동합니다.

### 🟡 `SafeZone_Check.py` (Safety First)
로봇의 **고장 및 사고를 방지**하는 진단 도구입니다.
* **자해 방지**: 그리퍼가 몸체에 부딪히지 않도록 최소 반경(15cm)을 유지합니다.
* **바닥 보호**: 최소 높이(2cm) 이하로 내려가지 않도록 차단합니다.
* **후방 접근 제한**: 케이블 꼬임 방지를 위해 로봇 뒤쪽($X < 0$) 이동을 금지합니다.

### 🔵 `workspace.py` (Analysis)
로봇이 움직일 수 있는 **최대 한계 범위**를 스캔합니다.
* 1cm 간격으로 3D 공간을 전수 조사하여 도달 가능한 범위를 산출합니다.
* 계산된 결과는 새로운 동작을 설계할 때 기준점으로 사용됩니다.

## 🚀 4. 시작하기 (Quick Start)

### 1️⃣ 설치 환경
* **OS**: Jetson Nano (Ubuntu 기반)
* **Library**: `Arm_Lib` (Yahboom 제공 라이브러리)

### 2️⃣ 실행 순서
```bash
# 1. 안전 좌표인지 먼저 확인 (입력 모드)
python3 SafeZone_Check.py

# 2. 로봇의 가동 범위 확인
python3 workspace.py

# 3. 메인 데모(사각형 그리기) 실행
python3 IK.py
